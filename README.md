The next <i>k</i> lines show letters' codes in "letter: code" format. The last line shows the encoded string.

<b>Sample Input 1:</b>

a

<b>Sample Output 1:</b>

1 1

a: 0

0

<b>Sample Input 1:</b>

abacabad

<b>Sample Output 1:</b>

4 14

a: 0

b: 10

c: 110

d: 111

01001100100111
